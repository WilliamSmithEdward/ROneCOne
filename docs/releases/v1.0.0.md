# ROneCOne 1.0.0

ROneCOne 1.0.0 brings a broad C#-inspired programming surface to ordinary Excel VBA while keeping
the deployed runtime to one dependency-free class module.

## Highlights

- Mutable, specialized, concurrent-style, and immutable generic collection families
- Broad LINQ querying, grouping, joining, aggregation, set, conversion, and ordering operators
- Cooperative tasks with await-style flow, cancellation, progress, continuations, and completion
  sources
- DataTable, DataSet, DataView, relations, constraints, change tracking, and provider adapters
- Universal delegates, expression lambdas, typed events, and structured exceptions
- Six polished, macro-enabled living demos with formula-backed results

## Release evidence

Each of three fresh Microsoft 365 Excel processes passed 384 live assertions. The versioned
10,000-operation benchmark samples remained inside all release gates, with observed maxima of
0.121 seconds for delegate invocation, 0.289 seconds for filtering, and 0.855 seconds for
composite ordering. Static analysis reported zero diagnostics across the source tree, test
workbook, and all six demo workbooks. No Office or VBE popup was observed.

## Host contract

The runtime targets Windows x64 Microsoft 365 Excel and stays inside one Excel process. Tasks are
cooperative because Excel owns VBA and workbook COM state on one thread. ROneCOne does not require
runtime VBIDE access, generate source, install an add-in, launch another Excel application, or
transmit workbook data.

Import `ROneCOne.cls` to begin, or open a demo workbook to see each capability run in Excel.
SHA-256 checksums for every release asset are included with the download.
